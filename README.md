# WalkQuest / Plot Capture

**Walk. Explore. Conquer.**

A mobile-first territory conquest platform that gamifies real-world walking through competition, exploration, achievements, and community rankings.

## Tech Stack

**Backend:**
- Java 21
- Spring Boot 3.x
- Spring Security with JWT
- Spring Data JPA
- PostgreSQL
- Maven

**Frontend:**
- HTML5/CSS3
- JavaScript (ES6+)
- Bootstrap/Tailwind CSS
- Leaflet.js + OpenStreetMap
- Mobile First Design

## Features (MVP)

- User Authentication (Registration, Login, JWT)
- Real-time Walking Tracking
- Grid-Based Territory Capture
- XP & Leveling System
- Badge Achievements
- Streak Tracking
- Activity Calendar (LeetCode Style)
- Leaderboards (Area, City, State, India)
- Champion Titles
- Admin Dashboard
- Anti-Cheat Detection
- Privacy-First Design
